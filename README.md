# Frogger Game (C++)

Console-based **Frogger-style** game written in **C++**.

## Repository

Description: *Console-based Frogger-style game in C++*.

## Files

- `frog.cpp` — main source file.

## Build

No build scripts are included (no `Makefile` / `CMakeLists.txt`). You can compile with `g++`:

```bash
g++ -O2 -std=c++17 frog.cpp -o frogger
```

## Run

```bash
./frogger
```
